
### 2.3.5. 模型效果评估

从上文我们已经掌握了高效学得一个神经网络模型的方法，那么我们要如何评估这个模型的好坏呢？

ML 的学习与效果评估大多遵循这样的过程：

1. 数据划分：将能获取的`数据集` D 划分为`训练集` S，`验证集` V，`测试集` T；
2. `算法`和`超参数`选择：通过 S 进行不同算法和超参数的模型训练，并通过 V 进行效果验证，选择效果最佳的算法和超参数；
3. 模型训练：通过 S + V 使用最佳的算法和超参数进行模型训练，学得理论上最好的模型；
4. 交付测试：通过 T 进行模型的泛化性能评估；最终找到泛化性能最好的模型；
5. 提交模型：泛化性能达标后，最终使用 D 重新训练该模型，得到最终模型；

其中，验证集（`validation set`）是进行算法与超参数选择时，用来评估模型好坏的数据集，用于模型选择和超参数的调参。测试集（`testing set`）则是未被学习过程使用过的，用来评估模型泛化性能的数据集。往往从样本真实分布中独立同分布采样而得，应尽量与训练集互斥。

最终模型是指学习算法和超参数已选定，此时应使用数据集 D 全集重新训练模型，这才是最终提交给用户的模型。

数据划分有专业的方法：

| 采用方式                | 英文名称                                 | 说明                                                                                                                                                                                                                |
| ----------------------- | ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 留出法                  | hold-out                                 | 将数据集 D 划分为两个互斥的集合，其中一个集合作为训练集 S，另一个作为测试集 T（D = S $\bigcup$ T; $\emptyset$ = S $\bigcap$ T)；                                                                                    |
| 分层采样                | stratified sampling                      | 保留类别比例的采样方式；                                                                                                                                                                                            |
| 交叉验证法/k 折交叉验证 | cross validation/k-fold cross validation | 将数据集 D 划分为 k 个大小相似，数据分布一致（通过分层采样获得）的互斥子集，每次用 k-1 个子集对病机作为训练集，剩余那个子集作为测试集；这样可以获得 k 组训练/测试集，进行 k 次训练和测试后，求 k 次测试结果的均值； |
| 留一法                  | Leave-One-Out, LOO                       | k 折交叉验证中的 k 取值与样本个数相同时，训练集与初始数据集相比只少了一个样本；                                                                                                                                     |
| 自助采样法              | bootstrap sampling                       | 给定包含 m 个样本的数据集 D，每次随机从 D 中挑选一个样本拷贝到 D' 中，然后将其放回 D，如此重复 m 次，D' 集合为采样结果；                                                                                            |
| 自助法                  | bootstrapping                            | 以自助采样法从数据集 D 采样得到数据集 D'，约有 36.8% 的样本未出现在 D' 中，将 D' 用作训练集，D\D' 用作测试集；                                                                                                      |
| 包外估计                | out-of-bagestimate                       | 使用没有在训练集中出现的样本作为测试集；                                                                                                                                                                            |

在数据集较小时适合自助法，可以使用原始数据集样本个数的样本作为训练集，但自助法会改变数据集的初始分布，引入估计偏差，数据量足够时更宜使用留出法和交叉验证法。

**性能度量指标参考**

| 评估指标            | 英文名称                          | 说明                                                                                                                                                                                                                                                                       |
| ------------------- | --------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 误差                | error                             | 模型的实际预测输出与样本的真实输出之间的差异；                                                                                                                                                                                                                             |
| 错误率              | E, error rate                     | 分类错误的样本占样本总数的比例, E = a/m；                                                                                                                                                                                                                                  |
| 准确率              | accuracy                          | 1 - E                                                                                                                                                                                                                                                                      |
| 查准率              | precision                         | 真正例占真正例和假正例之和的比例；                                                                                                                                                                                                                                         |
| 查全率/召回率       | recall                            | 真正例占所有正例（真正例和假反例）之和的比例；                                                                                                                                                                                                                             |
| 宏查准率            | macro-P                           | 基于混淆矩阵分别计算查准率，再求平均值；                                                                                                                                                                                                                                   |
| 宏查全率            | macro-R                           | 基于混淆矩阵分别计算查全率，再求平均值；                                                                                                                                                                                                                                   |
| 宏 F1               | macro-F1                          | 基于宏查准率和宏查全率计算的 F1；                                                                                                                                                                                                                                          |
| 微查准率            | micro-P                           | 对混淆矩阵的对应元素进行平均，得到 TP、FP、TN、FN 的均值，再计算查准率；                                                                                                                                                                                                   |
| 微查全率            | micro-R                           | 对混淆矩阵的对应元素进行平均，得到 TP、FP、TN、FN 的均值，再计算查全率；                                                                                                                                                                                                   |
| 微 F1               | micro-F1                          | 基于微查准率和微查全率计算的 F1；                                                                                                                                                                                                                                          |
| P-R 曲线            |                                   | 将样例按预测为正例的概率高低排序，然后逐个把样本作为正例进行预测评估，计算其查准率和查全率，以查全率为横轴，查准率为纵轴绘制一个曲线图；通常若学习器 A 的 P-R 曲线能完全包住学习器 B 的，则可断言 A 优于 B；若存在交叉，可采用面积比较法，或平衡点比较法；                 |
| 平衡点              | Break-Event Point, BEP            | 查准率 = 查全率 时的取值；                                                                                                                                                                                                                                                 |
| f1                  |                                   | 2 除以查全率和查准率的倒数之和；                                                                                                                                                                                                                                           |
| ROC                 | Receiver Operating Characteristic | 受试者工作特征， 以“真正例率”（True Positive Rate，TPR，TP / (TP + FN)，真的里面被测出来的）为纵轴，以“假正例率”（False Positive Rate，FPR，FP / (TN + FP)，假的里面被错认为真的）为横轴；与 P-R 图相似，若学习器 A 的曲线完全“包住”学习器 B 的曲线，则可可断言 A 优于 B； |
| AUC                 | Area Under ROC Curve              | ROC 曲线下的面积。当 ROC 曲线相交时，可通过 AUC 判定学习器的好坏；                                                                                                                                                                                                         |
| 分类阈值            | threshold                         | 将学习器为测试样本产生的实值或概率预测安大小排序，然后通过某个“截断点”（cut point）将样本分为两部分，前一部分为正例，后一部分为反例；                                                                                                                                      |
| 非均等代价          | unequal cost                      | 权衡不同类型错误所造成的不同损失；                                                                                                                                                                                                                                         |
| 总体代价            | total cost                        | 不同错误按非均等代价累计的代价之和；                                                                                                                                                                                                                                       |
| 代价曲线            | cost curve                        | 可通过其下面积计算总体代价，通过 ROC 曲线上的点与之相应的 FNR 切取；                                                                                                                                                                                                       |
| 训练误差 / 经验误差 | training error / empirical error  | 模型在训练集上的误差；                                                                                                                                                                                                                                                     |
| 泛化误差            | generalization error              | 在新样本上的误差；我们希望得到泛化误差小的模型，而我们实际能做的事努力使经验误差最小化；                                                                                                                                                                                   |
| 过拟合              | overfitting                       | 把训练样本自身的一些特点当作了所有潜在样本都会具有的一般性质；                                                                                                                                                                                                             |
| 欠拟合              | underfitting                      | 对训练样本的一般性质尚未学好；                                                                                                                                                                                                                                             |
| 置信度              | confidence                        | 种概率性度量，用于表示模型对一个预测结果的确定程度；在分类问题中，置信度可以理解为模型预测某个样本属于某个类别的概率；在回归问题中，置信度可以被定义为误差范围的估计值或标准差等度量；                                                                                     |
