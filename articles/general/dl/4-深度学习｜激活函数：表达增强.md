## 引言

在前文对 M-P 神经元结构的介绍以及「深度学习｜模型推理：端到端任务处理」的推理过程演示中，我们反复提到了`激活函数` $\sigma()$ 的概念，它决定了神经元的输出，在神经网络中起着至关重要的作用。激活函数的选择直接影响模型的性能和学习能力。

激活函数是一种用于引入非线性特性的数学函数，用于将加权和转换为更复杂的输出，使神经网络能够学习复杂的模式。通过引入非线性特性，可以解决非线性问题（否则神经网络的输出只是输入的线性组合，无法处理复杂的非线性问题），同时增强网络的表达能力，让其可以处理更复杂的问题。

> 激活函数对**梯度的传播**也很重要，一些激活函数（如 `ReLU`）具有`稀疏激活`性质，可以避免`梯度消失问题`，有利于更好的传播误差和加快`收敛`速度等作用。这些好的性质我们都将在后文对深度学习的进一步探讨中呈现给大家。

## 常见的激活函数

激活函数的选择往往根据求解问题的性质决定，一般回归问题可以使用`恒等函数`，二分类问题可以使用 `sigmoid` 函数，多分类问题可以使用 `softmax` 函数。

前文的手写数字识别任务中，输出层与隐层所用的激活函数 $\sigma$() 就有所不同。隐层使用了 `ReLU` 函数，输出层使用了 `softmax` 函数，我们将这些激活函数一一展开介绍。

### **Sigmoid 函数**

**公式**

$$
  f(x) = \frac{1}{1 + e^{-x}}
$$

**特点**

- 输出范围在 (0, 1) 之间，易于理解和解释。
- 可以用于二分类问题。
- 容易导致梯度消失（vanishing gradient）问题，影响深层网络的训练。

### **双曲正切函数 (Tanh)**

**公式**

$$
  f(x) = \tanh(x) = \frac{e^{x} - e^{-x}}{e^{x} + e^{-x}}
$$

**特点**

- 输出范围在 (-1, 1) 之间，通常比 Sigmoid 函数更好。
- 仍然可能受到梯度消失的问题，但影响较小。

### **修正线性单元 (ReLU)**

**公式**

$$
  f(x) = \max(0, x)
$$

**特点**

- 在正区间中，输出是线性的，负区间输出为 0。
- 计算简单，收敛速度快，广泛使用。
- 可能导致“死亡 ReLU”问题，即一些神经元在训练过程中始终输出 0。

### **Leaky ReLU**

**公式**

$$
  f(x) = \begin{cases}
  x & \text{if } x > 0 \\
  \alpha x & \text{otherwise}
  \end{cases}
$$

其中 $\alpha$ 是一个小的常数（例如 0.01）。

**特点**

- 解决了 ReLU 的“死亡”问题，在负区间提供了一个小的线性输出。

### **Softmax 函数**

**公式**

$$
  f(x_i) = \frac{e^{x_i}}{\sum_{j} e^{x_j}}
$$

**特点**

- 将输入转换为概率分布，常用于多分类问题的输出层。
- 输出值在 (0, 1) 之间，且所有输出的和为 1。

#### 1. 阶跃函数

`阶跃函数`以阈值为界，输入超过阈值则切换输出。如式 2.1.2.3 所示就是阶跃函数，它以 0 为界，小于 0 时输出 0，超过 0 时输出 1。

感知机使用阶跃函数作为激活函数，因此感知机的神经元之间流动的只能是 0 或 1 的二元信号。

#### 2. sigmoid 函数

`sigmoid 函数`是一条形如图 2.2.3.2 的平滑曲线，其平滑性可以返回 0 至 1 之间的实数，数学式表示为式 2.2.3.1。

<!-- <div align=center>
    <img src='../images/ann/2.2.3-sigmoid.png' width=500><br/>
    图 2.2.3.2 sigmoid 函数
</div><br/> -->

$$
    h(x) = \frac{1}{1 + e^{-x}}             \tag{2.2.3.1}
$$

神经网络的隐层常使用 sigmoid 函数作为激活函数，这使得神经元之间流动的是连续的实数值信号，这对神经网络的 BP 算法学习具有重要意义。

阶跃函数与 sigmoid 函数的共同点是，当输入信号较小时（不重要的信号），输出接近 0，输入变大时（重要的信号），输出接近于 1，输出信号都介于 0 至 1 之间。

阶跃函数与 sigmoid 函数都是非线性函数，这也是激活函数的基本要求，若使用线性函数做激活函数，将无法发挥多层神经网络的优势。比如三层神经网络使用线性函数作为激活函数，其运算可近似用 $y = c \times c \times c \times x$ 来表示，那与用 $y = a \times x$ 表示的单层网络是等效的。

#### 3. ReLU 函数

`ReLU`（`Rectified Linear Unit`）函数，在输入大于 0 时，直接输出该值，输入小于等于 0 时直接输出 0。ReLU 激活函数主要用于避免神经网络学习过程中的梯度消失与梯度爆炸，其定义为式 2.2.3.2。

$$
    h(x) = \begin{cases}
        x & (x > 0) \\
        0 & (x \leq 0)
    \end{cases}                            \tag{2.2.3.2}
$$

#### 4. 恒等函数

恒等函数将输入按原样输出，常用于回归问题。

<!-- <div align=center>

#### 5. Softmax 函数

`Softmax 函数`将 n 个元素转换成和为 1，取值在 0.0 到 1.0 之间的概率分布表示，用于计算这 n 个元素中，第 k 个元素的概率。其定义为式 2.2.3.3。

$$
    y_k = \frac{e^{a_k}}{\sum_{i=1}^{n}e^{a_i}}   \tag{2.2.3.3}
$$

<!-- <div align=center>
    <img src='../images/ann/2.2.3-softmax.png' width=500><br/>
    图 2.2.3.5 Softmax 函数
</div><br/> -->

softmax 函数多用于多分类任务的神经网络学习阶段的输出层激活函数，一般输出时只取 softmax 的值最大（最大概率）的项。推理阶段则可以省略 softmax 的计算。

### 3. 激活函数的选择

- 不同的任务和网络结构可能适合不同的激活函数。例如：
  - **Sigmoid** 和 **Tanh** 通常用于较小的网络或较浅的网络。
  - **ReLU** 和 **Leaky ReLU** 常用于深层神经网络，因为它们有助于加速收敛并降低梯度消失的问题。
  - **Softmax** 用于多分类输出层。

### 4. 未来趋势

- 研究者不断探索新的激活函数，力求提高模型性能，例如 Swish、Mish 等新兴激活函数。
- 激活函数的选择不断结合科学研究和实证经验进行优化，以提高模型的训练效果和泛化能力。

### 总结

激活函数是神经网络的核心组件之一，选择合适的激活函数对于模型的性能至关重要。随着深度学习的发展，激活函数的类型、性质及其设计原则也在不断演进。了解各种激活函数的特性，有助于设计更有效的神经网络结构。

下面是不同激活函数的 Python 实现，包括单个数据的计算和批量数据的计算，同时使用 `matplotlib` 可视化这些激活函数的图形。

### 激活函数实现和可视化

```python
import numpy as np
import matplotlib.pyplot as plt

# 激活函数实现

# 1. Sigmoid 函数
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def sigmoid_single(x):
    return 1 / (1 + np.exp(-x))

# 2. 双曲正切函数 (Tanh)
def tanh(x):
    return np.tanh(x)

def tanh_single(x):
    return np.tanh(x)

# 3. ReLU (修正线性单元)
def relu(x):
    return np.maximum(0, x)

def relu_single(x):
    return np.maximum(0, x)

# 4. Leaky ReLU
def leaky_relu(x, alpha=0.01):
    return np.where(x > 0, x, alpha * x)

def leaky_relu_single(x, alpha=0.01):
    return x if x > 0 else alpha * x

# 5. Softmax 函数
def softmax(x):
    e_x = np.exp(x - np.max(x))  # 减去最大值以提高数值稳定性
    return e_x / e_x.sum(axis=0)

def softmax_single(x):
    e_x = np.exp(x - np.max(x))  # 减去最大值以提高数值稳定性
    return e_x / e_x.sum()

# 创建输入数据
x_values = np.linspace(-10, 10, 400)
y_sigmoid = sigmoid(x_values)
y_tanh = tanh(x_values)
y_relu = relu(x_values)
y_leaky_relu = leaky_relu(x_values)
y_softmax = softmax(np.array([x_values]))  # Softmax on batch data

# 可视化所有激活函数
plt.figure(figsize=(12, 8))

# Sigmoid 图
plt.subplot(231)
plt.plot(x_values, y_sigmoid, label='Sigmoid', color='blue')
plt.title('Sigmoid Function')
plt.grid()
plt.xlabel('Input')
plt.ylabel('Output')

# Tanh 图
plt.subplot(232)
plt.plot(x_values, y_tanh, label='Tanh', color='orange')
plt.title('Tanh Function')
plt.grid()
plt.xlabel('Input')
plt.ylabel('Output')

# ReLU 图
plt.subplot(233)
plt.plot(x_values, y_relu, label='ReLU', color='green')
plt.title('ReLU Function')
plt.grid()
plt.xlabel('Input')
plt.ylabel('Output')

# Leaky ReLU 图
plt.subplot(234)
plt.plot(x_values, y_leaky_relu, label='Leaky ReLU', color='red')
plt.title('Leaky ReLU Function')
plt.grid()
plt.xlabel('Input')
plt.ylabel('Output')

# Softmax 图
plt.subplot(235)
plt.plot(x_values, y_softmax.T)  # 转置以适应绘图
plt.title('Softmax Function')
plt.grid()
plt.xlabel('Input')
plt.ylabel('Probability')
plt.legend([f'Class {i}' for i in range(y_softmax.shape[0])], loc='upper left')

plt.tight_layout()
plt.show()
```

### 代码说明

1. **函数实现**：

   - 每个激活函数都有对应的单个数据计算函数（如 `sigmoid_single`）和批量数据计算函数（如 `sigmoid`）。
   - Softmax 函数特别对待，因为它通常用在多分类问题中，因此对输入数组进行批量处理时需要考虑数值稳定性。

2. **输入数据**：

   - 创建一个从 -10 到 10 的线性空间，用于性能测试和可视化。

3. **可视化**：
   - 使用 `matplotlib` 绘制每一个激活函数的图形。
   - 在不同的子图中显示每个激活函数的输出。

### 运行结果

运行上述代码后，你将看到五个子图，每个子图展示了 Sigmoid、Tanh、ReLU、Leaky ReLU 和 Softmax 函数的输出。这可以帮助你更好地理解每个激活函数的方程和特点。

下面是对所有激活函数的横向和纵向分析对比，其中包括每个激活函数的特点、优缺点和适用场景。

### 横向分析对比表

| 激活函数       | 输出范围       | 计算复杂度 | 梯度问题                      | 可训练性 | 适用场景                     |
| -------------- | -------------- | ---------- | ----------------------------- | -------- | ---------------------------- |
| **Sigmoid**    | (0, 1)         | 低         | 梯度消失 (vanishing gradient) | 有限     | 二分类问题                   |
| **Tanh**       | (-1, 1)        | 低         | 梯度消失 (vanishing gradient) | 有限     | 二分类问题，特别是中心化数据 |
| **ReLU**       | [0, ∞)         | 低         | 死亡 ReLU 问题                | 高       | 深层网络，图像识别任务       |
| **Leaky ReLU** | (-∞, ∞)        | 低         | 减少死亡 ReLU 问题            | 高       | 深层网络，图像识别任务       |
| **Softmax**    | (0, 1)且和为 1 | 中         | 没有                          | 高       | 多分类问题的输出层           |

### 纵向分析

1. **输出范围**

   - **Sigmoid**: 输出范围在 (0, 1)，适用于需要概率输出的情况。
   - **Tanh**: 输出在 (-1, 1) 之间，更适合对称数据。
   - **ReLU**: 输出范围从 0 到无穷大，处理非线性特征非常有效，但负值输出为 0。
   - **Leaky ReLU**: 会为负输入提供小的非零输出，解决了 ReLU 的某些局限性。
   - **Softmax**: 输出概率分布，合适于多分类问题。

2. **计算复杂度**

   - 所有激活函数都具有较低的计算复杂度，尤其是 ReLU、Leaky ReLU，相比于 Sigmoid 和 Tanh 更容易被实现。

3. **梯度问题**

   - **Sigmoid** 和 **Tanh** 都可能遭遇梯度消失的问题，导致网络训练困难。
   - **ReLU** 可能导致一些神经元“死亡”，不再更新。
   - **Leaky ReLU** 通过避免完全“死亡”来缓解这一问题。
   - **Softmax** 本身没有梯度问题，但在与交叉熵损失函数结合使用时，能更有效地训练模型。

4. **可训练性**

   - Sigmoid 和 Tanh 在深层网络中可能会导致参数的更新减缓，影响可训练性。
   - ReLU 和 Leaky ReLU 显示出较高的可训练性，因其快速的梯度更新。

5. **适用场景**
   - **Sigmoid** 和 **Tanh** 适合于较小的网络或特定场景，需要概率输出。
   - **ReLU** 和 **Leaky ReLU** 被广泛推荐用于深层网络，特别是在图像识别等任务中。
   - **Softmax** 被用于分类模型的输出层，以确保最终输出为概率分布。

### 总结

不同的激活函数在输出范围、计算复杂度、梯度表现和适用场景等方面存在显著差异。选择合适的激活函数应根据具体任务的需求、网络结构与数据特性进行考量。在实际应用中，ReLU 和其变种（如 Leaky ReLU）因具备较好的性能，通常是深度神经网络中活跃的选择。
