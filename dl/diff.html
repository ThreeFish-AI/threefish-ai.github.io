求导公式：梯度逆传播规律

信息
2024年9月25日 · ·

引言​

我们知道，神经网络的能够学习处理任务的核心是计算损失的梯度，而误差逆传播算法是求梯度的一种通用且高效的办法。使用误差逆传播算法求解神经网络的梯度，其实就是求网络中使用的各种基本运算的局部导数的过程。这期我们将回顾各类基本运算的求导公式，然后演示如何将这些公式运用在误差逆传播算法求解网络梯度。

基础函数​

常数函数​

常数函数是指函数的值在定义域内保持不变，即：

f(x)=c f(x) = cf(x)=c

其中 ccc 是常数。

常数函数的导数为零：

ddx[c]=0 \frac{d}{dx}[c] = 0dxd​[c]=0

使用 Python 实现常数函数及其导数：

def constant_function(c):
    return lambda x: np.full_like(x, c)


def constant_derivative(c):
    return lambda x: np.zeros_like(x)

查看常数函数及其导数的图形：

# 绘制常数 c=1 时，常数函数的图形、常数函数在 x=1.0 处的切线、常数函数的导数图形
draw(x, constant_function, constant_derivative, "Constant Function(c=1)", x_point=1.0, color='blue', c=1)

常数函数的图形是一条截距为 ccc 的水平直线，其导数为零，导数函数的图形是一条全为零的水平直线。

幂函数​

幂函数是指函数的定义域为实数，求关于 xxx 的 nnn 次幂的函数，即：

f(x)=xn f(x) = x^nf(x)=xn

其中 nnn 是整数。

幂函数 f(x)=xnf(x) = x^nf(x)=xn 的导数为：

ddx[xn]=nxn−1 \frac{d}{dx}[x^n] = nx^{n-1}dxd​[xn]=nxn−1

使用 Python 实现幂函数及其导数：

def power_function(n):
    return lambda x: np.power(x, n)


def power_derivative(n):
    return lambda x: n * np.power(x, n - 1)

ddx[ln⁡(x)]=1x \frac{d}{dx}[\ln(x)] = \frac{1}{x}dxd​[ln(x)]=x1​

一般对数 log⁡a(x)\log_a(x)loga​(x) 的导数：

ddx[log⁡a(x)]=1xln⁡(a) \frac{d}{dx}[\log_a(x)] = \frac{1}{x \ln(a)}dxd​[loga​(x)]=xln(a)1​

ddx[log⁡10(x)]=1xln⁡(10) \frac{d}{dx}[\log_{10}(x)] = \frac{1}{x \ln(10)}dxd​[log10​(x)]=xln(10)1​

三角函数​

正弦函数 sin⁡(x)\sin(x)sin(x) 的导数：

ddx[sin⁡(x)]=cos⁡(x) \frac{d}{dx}[\sin(x)] = \cos(x)dxd​[sin(x)]=cos(x)

余弦函数 cos⁡(x)\cos(x)cos(x) 的导数：

ddx[cos⁡(x)]=−sin⁡(x) \frac{d}{dx}[\cos(x)] = -\sin(x)dxd​[cos(x)]=−sin(x)

正切函数 tan⁡(x)\tan(x)tan(x) 的导数：

ddx[tan⁡(x)]=sec⁡2(x) \frac{d}{dx}[\tan(x)] = \sec^2(x)dxd​[tan(x)]=sec2(x)

反三角函数​

反正弦函数 arcsin⁡(x)\arcsin(x)arcsin(x) 的导数：

ddx[arcsin⁡(x)]=11−x2 \frac{d}{dx}[\arcsin(x)] = \frac{1}{\sqrt{1-x^2}}dxd​[arcsin(x)]=1−x2​1​

反余弦函数 arccos⁡(x)\arccos(x)arccos(x) 的导数：

ddx[arccos⁡(x)]=−11−x2 \frac{d}{dx}[\arccos(x)] = -\frac{1}{\sqrt{1-x^2}}dxd​[arccos(x)]=−1−x2​1​

反正切函数 arctan⁡(x)\arctan(x)arctan(x) 的导数：

ddx[arctan⁡(x)]=11+x2 \frac{d}{dx}[\arctan(x)] = \frac{1}{1+x^2}dxd​[arctan(x)]=1+x21​

双曲函数​

双曲正弦函数 sinh⁡(x)\sinh(x)sinh(x) 的导数：

ddx[sinh⁡(x)]=cosh⁡(x) \frac{d}{dx}[\sinh(x)] = \cosh(x)dxd​[sinh(x)]=cosh(x)

双曲余弦函数 cosh⁡(x)\cosh(x)cosh(x) 的导数：

ddx[cosh⁡(x)]=sinh⁡(x) \frac{d}{dx}[\cosh(x)] = \sinh(x)dxd​[cosh(x)]=sinh(x)

双曲正切函数 tanh⁡(x)\tanh(x)tanh(x) 的导数：

ddx[tanh⁡(x)]=sech2(x) \frac{d}{dx}[\tanh(x)] = \text{sech}^2(x)dxd​[tanh(x)]=sech2(x)

双曲函数是在数学中类似于常见三角函数的一类函数，用于描述双曲线的几何性质。双曲函数按照与三角函数相似的方式，但专注于双曲线的性质，描述许多自然现象。双曲函数在物理和工程中有广泛应用，如描述悬链线问题等。它们与指数函数有密切的关系，同时也满足许多类似于三角函数的恒等式。

复合函数​

链式法则​

ddx[f(g(x))]=f′(g(x))⋅g′(x) \frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)dxd​[f(g(x))]=f′(g(x))⋅g′(x)

这些是微积分中常用的基本求导规则，可用于多种函数组合。

函数相加​

加 / 减法 函数​

加 / 减法是指将两个函数相加 / 减得到一个新的函数，即：

f(x)=g(x)±h(x) f(x) = g(x) \pm h(x)f(x)=g(x)±h(x) ddx[f(x)±g(x)]=f′(x)±g′(x) \frac{d}{dx}[f(x) \pm g(x)] = f'(x) \pm g'(x)dxd​[f(x)±g(x)]=f′(x)±g′(x)

乘法​

ddx[f(x)g(x)]=f′(x)g(x)+f(x)g′(x) \frac{d}{dx}[f(x)g(x)] = f'(x)g(x) + f(x)g'(x)dxd​[f(x)g(x)]=f′(x)g(x)+f(x)g′(x)

除法​

ddx[f(x)g(x)]=f′(x)g(x)−f(x)g′(x)[g(x)]2 \frac{d}{dx}\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}dxd​[g(x)f(x)​]=[g(x)]2f′(x)g(x)−f(x)g′(x)​

函数相减​

函数相差​

ddx[f(x)g(x)]=f′(x)g(x)+f(x)g′(x) \frac{d}{dx}[f(x)g(x)] = f'(x)g(x) + f(x)g'(x)dxd​[f(x)g(x)]=f′(x)g(x)+f(x)g′(x)

函数相除​

ddx[f(x)g(x)]=f′(x)g(x)−f(x)g′(x)[g(x)]2 \frac{d}{dx}\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}dxd​[g(x)f(x)​]=[g(x)]2f′(x)g(x)−f(x)g′(x)​

在微积分中，进行除法求导时，使用的是商的求导法则。对于两个可导函数 u(x)u(x)u(x) 和 v(x)v(x)v(x)，其商 u(x)v(x)\frac{u(x)}{v(x)}v(x)u(x)​ 的导数 (uv)′\left(\frac{u}{v}\right)'(vu​)′ 计算如下：

(uv)′=u′v−uv′v2 \left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}(vu​)′=v2u′v−uv′​

公式解释

  • u′u'u′ 是 uuu 对 xxx 的导数。
  • v′v'v′ 是 vvv 对 xxx 的导数。
  • v2v^2v2 是 vvv 的平方，确保分母不为零。

求导步骤

  1. 导出分子 uuu 的导数 u′u'u′。
  2. 导出分母 vvv 的导数 v′v'v′。
  3. 应用公式计算：u′v−uv′v2\frac{u'v - uv'}{v^2}v2u′v−uv′​。

这个法则适用于分数形式的函数求导，是微积分中的一个基本技巧。

Softmax 函数​

Softmax 函数常用于多分类问题的输出层，在逆传播中，我们需要计算损失函数对 Softmax 输入的梯度。Softmax 的逆传播过程如下：

对于输入向量 z=[z1,z2,…,zn]\mathbf{z} = [z_1, z_2, \ldots, z_n]z=[z1​,z2​,…,zn​]，Softmax 输出 y=[y1,y2,…,yn]\mathbf{y} = [y_1, y_2, \ldots, y_n]y=[y1​,y2​,…,yn​] 定义为：

yi=ezi∑j=1nezjy_i = \frac{e^{z_i}}{\sum_{j=1}^{n} e^{z_j}}yi​=∑j=1n​ezj​ezi​​

逆传播步骤​

  1. 计算损失函数的梯度：

    • 假设损失函数为交叉熵损失，定义为：

      L=−∑i=1ntilog⁡(yi) L = -\sum_{i=1}^{n} t_i \log(y_i)L=−i=1∑n​ti​log(yi​)

      其中，tit_iti​ 为实际的类别标签。

  2. 求 Softmax 输入的梯度：

    • 对于损失 LLL，我们需要求关于 z\mathbf{z}z 的偏导数。
    • 使用链式法则，首先计算 ∂L∂yi\frac{\partial L}{\partial y_i}∂yi​∂L​，然后计算 ∂yi∂zk\frac{\partial y_i}{\partial z_k}∂zk​∂yi​​。

  3. 计算偏导数：

    • 对于 i=ki = ki=k：

      ∂yi∂zi=yi(1−yi) \frac{\partial y_i}{\partial z_i} = y_i (1 - y_i)∂zi​∂yi​​=yi​(1−yi​)

    • 对于 i≠ki \neq ki=k：

      ∂yi∂zk=−yiyk \frac{\partial y_i}{\partial z_k} = -y_i y_k∂zk​∂yi​​=−yi​yk​

  4. 组合所有的梯度：

    • 最终得到损失失对 z\mathbf{z}z 的梯度：

      ∂L∂zk=∑i=1n∂L∂yi⋅∂yi∂zk=yk−tk \frac{\partial L}{\partial z_k} = \sum_{i=1}^{n} \frac{\partial L}{\partial y_i} \cdot \frac{\partial y_i}{\partial z_k} = y_k - t_k∂zk​∂L​=i=1∑n​∂yi​∂L​⋅∂zk​∂yi​​=yk​−tk​

    • 这表明损失函数的梯度简化为 Softmax 输出与实际标签之间的差，对于交叉熵损失来说。

这一过程使得计算 Softmax 层的梯度既有效又简洁，在深度学习框架中得到了广泛应用。

误差逆传播​

假设神经网络的最后一层输出是经过 softmax 函数的 yyy，我们可以定义 softmax 函数如下：

yj=ezj∑kezk y_j = \frac{e^{z_j}}{\sum_{k} e^{z_k}}yj​=∑k​ezk​ezj​​

其中 zzz 是神经网络最后一层的未归一化的输出。

在误差逆传播中，我们需要根据损失函数 LLL 的输出偏导数来更新权重。

1. 计算关于 z 的梯度​

使用链式法则，可以从 LLL 到 zzz 计算梯度：

∂L∂zj=∂L∂yj⋅∂yj∂zj \frac{\partial L}{\partial z_j} = \frac{\partial L}{\partial y_j} \cdot \frac{\partial y_j}{\partial z_j}∂zj​∂L​=∂yj​∂L​⋅∂zj​∂yj​​

2. 计算 ∂yj∂zj\frac{\partial y_j}{\partial z_j}∂zj​∂yj​​​

使用 softmax 的导数，我们可以得到：

∂yj∂zk=yj(δjk−yk) \frac{\partial y_j}{\partial z_k} = y_j (\delta_{jk} - y_k)∂zk​∂yj​​=yj​(δjk​−yk​)

其中 δjk\delta_{jk}δjk​ 是克罗内克（Kronecker）delta，等于 1 当 j=kj=kj=k，否则等于 0。

3. 最终梯度​

最终的通式为：

∂L∂zj=yj−tj \frac{\partial L}{\partial z_j} = y_j - t_j∂zj​∂L​=yj​−tj​

这意味着，你可以直接得到关于 zzz 的梯度：

∂L∂z=y−t \frac{\partial L}{\partial z} = y - t∂z∂L​=y−t

误差逆传播​

误差逆传播算法（Backpropagation）是训练人工神经网络的一种基本方法。它的主要目的是通过计算每个神经元的梯度，来优化网络的权重以使得输出误差最小化。以下是误差逆传播算法的主要步骤和概念：

  1. 权重更新：

    • 使用梯度下降或其变体（如随机梯度下降、动量法等）来更新权重。
    • 更新公式通常为：w = w - η * (∂E/∂w)，其中w是权重，η是学习率，∂E/∂w是误差对权重的偏导数。

误差逆传播算法是神经网络的学习核心，通过不断调整权重，提升模型在训练数据上的表现，从而有望在未见过的数据上也能准确预测。

通过逆传播算法，我们可以快速计算损失关于 SoftmaxWithLoss 层参数的梯度，这样继续逆向传播，就可以算法得到损失关于神经网络的权重参数的梯度。

交叉熵误差关于输出 yyy 的导数为：

∂L∂yi=−tiyi \frac{\partial L}{\partial y_i} = -\frac{t_i}{y_i}∂yi​∂L​=−yi​ti​​

对于所有 iii，这可以以向量的形式表示为：

∂L∂y=−ty \frac{\partial L}{\partial y} = -\frac{t}{y}∂y∂L​=−yt​
